# 대시보드 만들기
```
2019~23년 서울시 자전거 교통사고 다발지역
```
## 사용한 데이터셋
공공데이터 포털에서 제공하는 '전국교통사고다발지역표준데이터'를 활용했다.
사고연도가 2019~2023년, 사고유형이 '자전거', 사고지역위치가 '서울특별시'에 해당하는 데이터를 필터링하여 시각화했다.
